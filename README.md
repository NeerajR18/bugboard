# bugboard
STM32 Based Development Board

Note : - This is still a work in progress.

bugboard is a development board with the STM32G431CBT6 as its core. 

This development board is intended to be used in a keyboard PCB. 

# Features

- STM32G431CBT6
- Built-in USB-C Port
- Small size design
